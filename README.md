# EggShooting
1. Cách cài đặt:

- Clone project trên github
- Vào folder STM32CubeIDE
- Chạy file .project
- Nhấn File -> Clean Project.
- Connect kit STM32F429ZIT6 với máy tính, chọn Debug Configuration, chọn đúng serial của Kit và Run.
